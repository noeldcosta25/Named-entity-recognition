# Named Entity Recognition (NLP)

This project extracts important entities from text using spaCy.

## Entities Detected
- Person
- Organization
- Location
- Date

## Example
Input: Rohit joined Google in 2022 at Bangalore

Output:
Rohit -> PERSON
Google -> ORG
2022 -> DATE
Bangalore -> GPE

## 
Built for educational purposes to understand pre-trained NER models using spaCy, not for real-world deployment.
